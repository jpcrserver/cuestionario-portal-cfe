GUÍA DE INSTALACIÓN - CUESTIONARIO PORTAL CFE

1. ¿QUÉ NECESITAS?
   - Un editor de texto (como Notepad++, VSCode)
   - Un servidor web (gratis: GitHub Pages, Netlify, Vercel, etc.)
   - Un navegador web moderno

2. PASOS PARA EJECUTARLO EN TU SERVIDOR WEB

   a. Descarga todos los archivos en una carpeta:
      - index.html
      - App.jsx
      - guia.txt (este archivo)

   b. Sube la carpeta a tu servidor web:
      - GitHub Pages (recomendado para principiantes):
        1. Crea una cuenta en GitHub
        2. Crea un nuevo repositorio vacío
        3. Sube los archivos
        4. Ve a Settings > Pages > Source: "main" branch > Guarda
        5. Tu sitio estará en https://tu-usuario.github.io/nombre-repo/ 

      - Netlify (muy fácil):
        1. Ingresa a https://www.netlify.com/drop/ 
        2. Arrastra y suelta la carpeta completa
        3. Netlify te dará una URL automática

   c. Si usas tu propio dominio:
      - Copia todos los archivos al directorio raíz de tu servidor web

3. FUNCIONAMIENTO
   - El cuestionario funciona completamente en el navegador
   - No requiere servidor backend ni base de datos
   - Todos los datos se validan en el frontend

4. EDICIÓN DEL FORMULARIO
   - Abre App.jsx y modifica:
     - Campos
     - Valores por defecto
     - Validaciones
   - Guarda los cambios y vuelve a subir los archivos

5. SOPORTE TÉCNICO
   - Si ves errores en consola:
     - Asegúrate de usar HTTPS
     - Verifica que todos los archivos se carguen correctamente
     - Usa DevTools del navegador para debuggear
